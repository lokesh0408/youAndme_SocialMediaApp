## Installation

1. Clone the repository.
2. Install the frontend dependencies:
    ```bash
    npm install
    ```
3. Install the backend dependencies:
    ```bash
    npm install
    ```
    
## Usage
1. Start the server:
    ```bash
    npm start
    ```
2. Start the frontend development server:
    ```bash
    npm run dev
    ```
